SimplyWeb is a lightweight PHP framework.
This is built to be played on and tested - can be used for basic web applications.
However we do not recommend you use this in production as this is not a proper product - but rather
a play thing I (Surden) has came up with. It is similar to Laravel - but missing a lot of features. 

Requirements:
 - PHP 8.2 and higher (up to 8.5), any PHP version over 8.0 should still work.
 - Ability to make /public the web root folder. 

You can read on the documentation for Simply-Web here at: https://surden.me/documentation/simply-web-getting-started/

Currently available built-in modules:
 - MVC (Model View Controller) model
 - Templating Engine
 - Email Service (Simply SMTP)
 - API Manager (create new API routes easily)
 - Accounts (Middleware)
 - Simple Domain-Whitelist feature
